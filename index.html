How to Get Your GitHub Token

Step-by-Step Instructions:

  1. Go to GitHub.com and sign in
  2. Click your profile picture ‚Üí Settings
  3. Scroll to the bottom ‚Üí Developer settings
  4. Click Personal access tokens ‚Üí Tokens (classic)
  5. Click Generate new token ‚Üí Generate new token (classic)
  6. Give it a name like "Clipboard Sync"
  7. Set expiration (or "No expiration")
  8. Check ONLY the gist checkbox
  9. Click Generate token
  10. IMPORTANT: Copy the token immediately!

How to Get Your Pantry ID

Quick Setup:

  1. Go to getpantry.cloud
  2. Enter your email address
  3. Choose a Pantry name (anything works - we'll create a basket called "clip_sync" inside it)
  4. You'll receive a Pantry ID (looks like: 12a34567-8901-2345-b678-9ce0123456e7)
  5. Important: Save this ID somewhere safe (password manager, notes, etc.)
  6. Copy and paste that ID here
  7. Use the same ID on all your devices
